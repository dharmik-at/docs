## What is IngestIQ?

IngestIQ is a comprehensive **Retrieval-Augmented Generation (RAG)** platform that enables AI application developers to easily integrate enterprise data sources without building RAG infrastructure from scratch.

## Key Features

<AccordionGroup>
  <Accordion title="Multi-format Document Processing" icon="file-lines">
    Process PDFs, CSV, Excel, and video content with intelligent parsing powered by Google Gemini AI.
  </Accordion>
  
  <Accordion title="Semantic Chunking" icon="scissors">
    AI-powered document chunking that understands context and meaning, not just character counts.
  </Accordion>
  
  <Accordion title="Vector Search" icon="magnifying-glass">
    High-performance semantic search using OpenAI embeddings and PostgreSQL with pgvector.
  </Accordion>
  
  <Accordion title="MCP Integration" icon="puzzle-piece">
    Native Model Context Protocol (MCP) server for seamless AI tool integration.
  </Accordion>
  
  <Accordion title="Scheduled Pipelines" icon="clock">
    Automated data ingestion with cron-based scheduling via BullMQ.
  </Accordion>
  
  <Accordion title="Multi-tenant Architecture" icon="building">
    Organizations with role-based access control for enterprise deployments.
  </Accordion>
</AccordionGroup>

## Data Connectors

IngestIQ supports multiple data sources out of the box:

| Connector | Description | Status |
|-----------|-------------|--------|
| **File Upload** | PDF, CSV, Excel direct uploads | Available |
| **Web Scraping** | Crawl4AI integration for web content | Available |
| **Google Drive** | OAuth2-based document sync | Available |
| **Video/YouTube** | FFmpeg + Whisper transcription | Available |
